## 关于项目

- **Hog**：一款结合策略与概率的掷骰子游戏模拟器。
- **Cats**：具备自动纠错功能的打字速度（WPM）与准确率测试软件。
- **Ants**：使用面向对象编程（OOP）实现的塔防游戏。
- **Scheme**：使用 Python 从零构建的 Scheme 语言解释器。
